  NongoTour Angola
</h1>

<img src="../main/NongoTour-Angola/files/pin_2377922.png" alt="rocket_nongotour" width="30" style="vertical-align: middle;"/>O **NongoTour**, é um aplicação web inteligente de recomendação de destinos turísticos em Angola, que com uso de **Machine Learning** permite aos turistas com base nas suas preferências a serem direccionados em diferentes sítios turísticos. 

O **NongoTour** vem da junção do ***Ubumdu*** e ***Inglês*** onde ***Noñgo*** *significa Inteligente* e ***Tour*** *significa Viagem*.

**MISSÃO:** *Ser o guia turístico inteligente de Angola, guiando visitantes a descobrirem as maravilhas do país através de uma plataforma web inteligente e personalizada.*
